# Hospital Management System

A Python-based Hospital Management System developed using Tkinter for GUI and SQLite for database management.

## Features
- Admin Login
- Patient Registration
- Staff Management
- Appointment Booking
- Billing System
- User-friendly GUI
- Image Slideshow

## Technologies Used
- Python
- Tkinter
- SQLite
- Pillow (PIL)
